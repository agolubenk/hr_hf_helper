Инструкция по настройке
1
Создайте интеграцию в Notion

Перейдите в Notion Integrations и создайте новую интеграцию

2
Скопируйте API токен

Скопируйте Internal Integration Token и вставьте в профиль пользователя

3
Добавьте интеграцию к базе данных

В Notion откройте нужную базу данных, нажмите "Share" и добавьте вашу интеграцию

4
Скопируйте ID базы данных

Из URL базы данных скопируйте ID (32-символьная строка) и вставьте в поле выше

5
Протестируйте подключение

Нажмите "Тестировать подключение" для проверки настроек

Полезная информация

ID базы данных: Находится в URL страницы базы данных Notion

Пример: https://notion.so/your-workspace/32-символьный-id?v=...

Права доступа: Интеграция должна иметь доступ к чтению базы данных
